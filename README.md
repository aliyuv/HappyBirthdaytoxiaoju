
# Xiaoju's Birthday H5

A premium mobile-first birthday greeting page.

## Getting Started

1. Install dependencies:
   ```bash
   pnpm i
   ```

2. Run development server:
   ```bash
   pnpm dev
   ```

## Features
- GSAP driven high-performance animations.
- Realistic CSS-only flames with flicker and glow effects.
- Interaction state machine for the birthday cake.
- Responsive starfield canvas background.
- Photo masonry with 3D tilt interaction.
